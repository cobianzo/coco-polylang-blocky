Extends the free Polylang plugin to insert content in the block editor adapted to each selected language. This plugin also adds the language_switcher block for Polylang.

== Description ==

Coco Polylang Blocky is a WordPress plugin that enhances the functionality of the Polylang plugin by providing seamless integration with the Gutenberg block editor. It allows you to display content based on the selected language and adds a convenient language switcher block.

Key Features:
* Display content based on the selected language.
* Adds a language switcher block for Polylang.
* Easy to use within the block editor.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/coco-polylang-blocky` directory
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Start using the new language-aware blocks in your content.

== Frequently Asked Questions ==

= How do I use the language-aware blocks? =

Simply insert the desired block and select the language you want to display it for. The content will be shown only when that language is active.

= Does this work with any theme? =

Yes, Coco Polylang Blocky works with any theme that supports the WordPress block editor (Gutenberg).
